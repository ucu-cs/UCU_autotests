# Running Tests
 
Most likely, you'd want to run the tests as:
```python
python test_1brc.py -b tests-build -n 1brc --data-suite <path-to-1brc_samples> --keep-build -l INFO
```
This compiles the `1brc` inside `tests-build` directory, and uses the pre-downloaded test samples. Run `python test_1brc.py --help` for the link or to explore other options.
